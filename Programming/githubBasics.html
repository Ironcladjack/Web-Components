Github Basics:

Setup

  1. CD into directory
  2. run 'git init' to create repository
  3. Check status of git repo with 'git status'
  4. Add files to staging areas with 'git add <code><filename>'
  5. Check difference between files in WD & SA with 'git diff <filename>'
  6. Add files to Repo with 'git commit -m ''Complete first line of dialogue'''
  7. View all previous commits 'git log'

Controlling Github Commits:

  1. Show most recent commit with 'git show HEAD'
  2. restore file to previous commit version with 'git checkout HEAD <filename> '
  3. remove files from staging area, if mistakenly added 'git reset HEAD <filename>'
  4. reset WD to a previous commit with 'git reset SHA' where 'SHA' is the first 7 characters in the commit SHA you want.

Controlling Branches:

  1. Show which branch we are working in with 'git branch'
  2. To create a new branch, use 'git branch <new_branch>'
  3. To switch to a branch, use 'git checkout <new_branch>'
  4. To merge a branch with another, navigate to host branch and use 'git merge <branch_name>'
  5. To remove a branch, use 'git branch -d branch_name'

Sharing Remotely:

  1. To clone an online repo, use 'git clone <remote_location> <clone_name>'
  2. You can see a list of a Git project's remotes with the command: 'git remote -v'
